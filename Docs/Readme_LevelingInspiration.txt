==============
1. DESCRIPTION
==============

Leveling Inspiration introduces Morrowind's and Oblivion's inspirational messages that show up once you gain a level.
Messages are shown until level 21.

Short. Simple. Sweet.


===============
2. INSTALLATION
===============

Either install using Vortex Mod Manager or install manually as described below.


Extract the content of the archive to the StreamingAssets folder of your Daggerfall Unity directory.

Example : [Your drive]\DagUnity\DaggerfallUnity_Data\StreamingAssets

If you already have the "Cliff" folder, you can overwrite without issue.


=============
3. SAVE GAMES
=============

If you are loading Leveling Inspiration on an existing save game, you will need to open Daggerfall Unity's console
and type the following : 

StartQuest LVLUP01


You will then receive every messages up to your current level.

The key to open the console can be found in the settings, under the "Advanced" tab in "Controls".
